# Расписание занятий

Android-приложение, созданное в рамках лабораторных работ по мобильной разработке.

## Описание

Приложение отображает расписание занятий и демонстрирует основные возможности Android-разработки с использованием Kotlin и Jetpack Compose.

## Реализованные возможности

- отображение списка занятий;
- добавление нового занятия;
- удаление занятия с подтверждением;
- отметка занятия через checkbox;
- экран подробной информации о занятии;
- навигация между экранами;
- экран "О приложении";
- экран настроек;
- сохранение выбранной темы оформления;
- поддержка светлой, тёмной и системной темы;
- интернет-запрос с обработкой загрузки и ошибки;
- экран разрешений:
    - камера;
    - микрофон;
    - GPS;
    - сенсоры;
- release-сборка приложения.

## Используемые технологии

- Kotlin
- Android Studio
- Jetpack Compose
- Material Design 3
- DataStore
- Ktor Client
- Git / GitHub

## Структура приложения

Приложение содержит несколько экранов:

- **Расписание** — основной экран со списком занятий.
- **Детали занятия** — экран с подробной информацией и интернет-запросом.
- **О приложении** — краткое описание приложения.
- **Настройки** — выбор темы оформления.
- **Разрешения** — просмотр и запрос разрешений устройства.

## Сборка и запуск

Для запуска проекта необходимо открыть его в Android Studio и нажать кнопку Run.

Для создания release-сборки используется:

```text
Build → Generate Signed App Bundle or APK → APK → release
```

Готовый APK находится по пути:

app/build/outputs/apk/release/app-release.apk
